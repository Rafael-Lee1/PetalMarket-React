## Run Locally

Clone the project

```bash
  git clone https://github.com/Rafael-Lee1/PetalMarket-React
```

Go to the project directory

```bash
  cd flower-for-thought-ecommerce-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
